Project

Package Structure

We organize the package into five,

  1. activity - All view shown on package ( com.example.app.cpux )
  2. adapter package - the image package included all support clas to activity, ( com.example.app.cpux )
  3. data packagae - included shared preference, ( com.example.app.cpux.data )
  4. model package -  included info model ( com.example.app.cpux.model )
  5. tools package -  any support for getting information ( com.example.app.musicplayer.tools)

Add Google Play LIbrary

Add google play library

This library needed before you use the admob in this application

1. import the google play services library into your workspace

2. right click your main project select properties

3. choose android tab in the left.

4. add the google play services library

5. ok

Ads

DMOB BANNER

for banner is appear in bottom of gallery activity. please change  the ID in res/value/string.xml admob banner

  1. <string name="banner_ad_unit_id">ca-app-pub-46199952.......</string>
 
interstitial ADS
this ads is appear in app after back from other activity or by press any button, change the ID in res/value/string.xml
  1. <string name="interstitial_ad_unit_id">ca-app-pub-4619995241......</string>
